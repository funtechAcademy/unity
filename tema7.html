Terrenos

Creamos un proyecto Universal 3D SRP(Universal Render Pipeline).
Necesitaremos varios Assets:
Package FirstPersonController (Lo facilitamos nosotros).
https://assetstore.unity.com/packages/3d/environments/low-poly-environment-nature-free-lowpoly-medieval -fantasy-series-187052.
https://assetstore.unity.com/packages/2d/textures-materials/sky/allsky-free-10-sky-skybox-set-146014.

Y en cada uno damos al botón de: Agregar a mis activos. Nos abre el Package Manager de Unity lo descargamos e importamos.

Y nos aparecera en Projects/Assets los 3 Assets que hemos descargado.

Crear un Terreno

Para crear un terreno en nuestra jerarquía damos click derecho 3D Object/Terrain y nos alejamos un poco para verlo desde más arriba con la rueda del ratón.

Montañas

• Seleccionar Paint Terrain.
• Y en el desplegable seleccionar Raise or Lower Terrain.

• Ajustar el Brush Size y el Opacity.

No pasaros con el Opacity porque las montañas quedarian muy escarpadas y puede quedar un poco raro.

Ir dibujando y cambiando de Brushes. Con el click izquierdo sube y si queremos bajar la montaña damos al SHIFT más el click izquierdo.

Texturas

Cambiamos el desplegable a Paint Texture.

Seleccionamos el botón Edit Terrain Layers/Add Layer y añadimos una nueva layer.

Y seleccionamos el verde.

Se puede ajustar el color en Color Tint.

Podemos crear una nueva Layer para añadiendo un poco de marrón. Y podemos ajustar el Brush Size y el Opacity para generar zonas más marrones.

Hierba

Añadir la hierba en Paint Details. Damos al botón Edit Details/Add Grass Texture.

Aparece un ventana pequeña.

Y en Detail Texture buscamos Grass y seleccionamos una.

Podemos ajustar el Detail density, el Max y Min Height del grass e incluso el color. Y le damos Add.

Nos acercamos y podemos empezar a poner hierba.

Si no lo veis bien tendreis que ajustar en Terrain Settings el Detail Distance. El botón que está al lado de Paint Details. Dependera también del ordenador para que vaya más rápido o despacio.

Árboles

Primero vamos a Polytope Studio/Lowpoly_Environments/URP y seleccionamos el último e importar.

Y ahora podemos añadir árboles en Paint Trees.

Edit Trees/Add Tree. Y buscar tree en la ventana pequeña que aparece. Se puede bajar el Tree Density y subir el Brush Size.

Vamos añadir otro tipo de árboles, roca, setas, troncos

Si la hierba está muy alta se puede reducir dando a Edit.

Cielo

Window/Rendering/Lighting/Enviroment:
• Environment/SkyBox Material seleccionamos el cielo.

Other Settings/Fog (niebla), podemos jugar con sus atributos.

Directional Light

• Podemos modificar su Rotación para cambiar las sombras.
• Cambiar la Temperatura al escenario.
• Y la Intensidad para que parezca que halla un poco más de sol.

Prefabs

Podemos añadir prefabs de rocas.. Polytope Studio/Lowpoly_Environments/Prefabs.

Main Camera

Seleccionar Main Camera para poder añadir efectos de Post Processing:
• Activar Post Processing

Global Volumen

Global Volumen haremos algunos cambios:
• Bloom aumentamos para que las cosas que tengan ilumnación brillen más.

• Podemos añadir en Add Override un Color Adjustments.

Esta dentro de Post-processing ó lo podemos buscar en la lupa.

• Podemos añadir en Add Override un Chanel Mixer para cambiar el color rojo para darle una tonalidad más calida.

• Podemos añadir en Add Override un Lift Gamma Gane para cambiar un poco el color de la escena. Se selecciona y se pincha en el centro y se arrastra.

Añadir al Player

Vamos a la carpeta donde este el Prefab Player y lo arrastramos a la escena:
• Subimos la Y para que no se quede debajo del terreno

• Eliminamos la Main Camera.
• Dentro de FirstPersonCharacter le añadimos el Post Processing.

Podriamos ejecutar y movernos dentro de nuestro escenario.

Existe otro paquete que es Terrain Tools que tendriais que instalar si quereis. Tiene más pinceles y es muy parecido.
Para crearlo Window/Terrain/Terrain Toolbox.

En nuestro ejemplo lo instale pero no cree ningún terreno.

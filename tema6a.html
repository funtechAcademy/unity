Cámaras

Este proyecto nos enseñara como tener varias cámaras y que podamos verlas a la vez.
Tendremos un Pollo que se moverá con las flechas y con el espacio para saltar hacia delante.

La cámara del juego que apunta solo al objeto Árbol que será un prefab y el Pollo tendrá una camára que verá todo menos al Árbol. Ambas cámaras serán Ortográficas.

Esto lo haremos mediante el Layer que tengan asignados los objetos.

Cuando el Árbol toque al Pollo desaparecerá y hara un sonido.

Cámaras

Tendremos 2 cámaras:
 • CámaraCentral apuntará solo al Layer Default y Personajes (es el prefab que creamos).
 • Cámara del Pollo verá todo menos el Layer de Personajes que la incluiremos cuando hagamos el Pollo.

Para poder decidir que trozo de Layer reproduce las cámaras usaremos un parámetro de la cámara: Culling Mask.

Suelo

• Será un plano con una textura.
• En el Layer ponemos “Terreno”.
• Componente AudioSource ponemos en AudioClip un sonido de golpe hit nada más empezar el juego.

Árbol

• Será una Cápsula.
• Puede ser atravesado por el Pollo.
• Lo haremos un prefab para poder poner varios.
• En el Layer ponemos “Personajes” para que la CameraCentral lo detecte.
• Tiene un Script Arbol.cs.
  o Cuando el Pollo choque con el Arbol se destruira al cabo de unos segundos.
  o Y hara un sonido.

Pollo

Pollo (Cubo) Mirando hacia la Z (adelante)
 • Modelo GO vacío.
  o Cuerpo.

No puede rotar en ningún eje.
Tiene un Script Pollo.cs.

Método EstoyEnSuelo() Lo puedo hacer de 2 maneras:
 • 1ª Manera:
  o Recoger el LayerMask.GetMask(“Terreno”).
  o Hacemos que se cree una una Esfera usando el método Physics.CheckSphere y parámetros de PosicionPatas, distancia y el layer del terreno y si da true es que estoy tocando suelo.
 • 2ª Manera:
  o Recojo todos los Collider en un array con Physics.OverlapSphere y si la longitud del array es Z 0 dara true y será que estoy tocando suelo.
 • Este método devolverá un booleano true si toca el suelo y false si no lo toca.

El CheckSphere devuelve un bool y el OverlapSphere devuelve un array con todos los colisionadores dentro del radio. Arbol.cs:

Árbol

Haremos un objeto para que cuando el Pollo se choque con él se destruira el objeto y hara un sonido. Le hacemos como Prefab para poner varios en el plano.
Tendrá un componente de Capsule Collider con el IsTrigger chequeado.

En el Layer del Arbol añadimos un Layer que se llame "Personajes".

Cuando tenemos el IsTrigger chequeado nos deja atravesar al objeto con el que colisionamos.

Tiene asociado un script Arbol.cs:
Start().
 • Recoger el AudioSource.
OnTriggerEnter().
 • Hacemos un sonido.
 • Y llamamos con el Invoke a un método DestruirObjeto() al cabo de 1 segundo que destruira el objeto.
